# 🏋️ SPORT PLUS

## Boutique e-commerce sportwear en PHP / MySQL

**SPORT PLUS** est une boutique e-commerce développée en **PHP natif + MySQL**, dédiée à la vente de **vêtements, chaussures et accessoires de sport**.

Le projet a été conçu pour être :

- moderne
- léger
- responsive
- facile à déployer sur **Hostinger**

---

## 📌 Présentation

SPORT PLUS est un projet e-commerce permettant de :

- présenter une boutique sportwear moderne
- afficher un catalogue produits
- gérer un panier et des favoris
- permettre la connexion / inscription client
- enregistrer des commandes
- proposer un espace administrateur

---

## 📝 Cahier des charges

Le site devait répondre aux besoins suivants :

- créer une **vitrine moderne**
- afficher un **catalogue produits**
- permettre l’**ajout au panier**
- gérer les **favoris**
- permettre la **commande en ligne**
- proposer un **suivi client**
- intégrer un **dashboard administrateur**
- rester **simple à maintenir**
- être **compatible Hostinger**

---

## ✨ Fonctionnalités
