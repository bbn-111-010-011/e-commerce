CHACHA V1 — PACK INTÉGRÉ BOUTIQUE + CONNEXION CLIENT

Pages principales :
- shop.php : page boutique d'accueil
- catalogue.php : catalogue
- produit.php?id=1 : fiche produit
- panier.php : panier
- login.php : connexion
- register.php : création de compte
- account.php : espace client

Important :
1. Placez ce dossier sur un serveur PHP (WAMP, XAMPP, Hostinger, etc.).
2. Importez votre base boutique_chacha.
3. Vérifiez config/database.php.
4. Ouvrez shop.php.

Cette version rassemble :
- chacha V1 boutique
- images locales produits
- tailles + quantité
- module inscription / connexion PHP
